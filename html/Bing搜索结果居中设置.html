Bing搜索结果居中设置

2024年01月12日 计算机 操作指南 字数统计: 145(字)

step 1 : 安装浏览器插件stylus

step 2 : 新建UserCss样式

抬头信息为

/* ==UserStyle==
@name         Center Bing Result
@version      20240111.23.46
@namespace    userstyles.world/user/dengfei
@description  Center the search result of Bing.
@author       Dengfei Zeng
@license      Mozilla Public License Version 2.0
==/UserStyle== */

主要内容为

@-moz-document regexp(".*bing.com/search.*") {

    #b_content {
        padding-left: 19%;
    }

    .b_searchboxForm {
        padding-left: 0px;
        margin-left: 18%;
    }

    #sb_form {
        width: 72%;
    }

    #est_switch {
        margin-left: 20%;
    }

    .b_scopebar {
        margin-left: 19%;
    }

    #b_results {
        width: 45%;
    }
}

注：如果因为网站更新失效，利用浏览器的检查可以重新修改。
