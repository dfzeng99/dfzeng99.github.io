Linux配置Git

2023年08月19日 计算机 操作指南 字数统计: 203(字)

一、安装Git

对于Debian类的Linux发行版，直接使用apt工具安装即可

sudo apt install git

二、配置Git

配置Git用户名和邮箱，运行如下命令：

git config --global user.name "zdenfo"
git config --global user.email "zengdengfei@163.com"

查看当前配置的命令为

git config --list

配置文件存储在~/.gitconfig中，需要修改Git配置时，可以使用git config或者直接修改配置文件。

三、配置ssh密钥

首先，运行如下命令生成SSH密钥

ssh-keygen -t rsa -C "zengdengfei@163.com"

提示的地方直接按Enter键跳过，然后查看生成的密钥

cat ~/.ssh/id_rsa.pub

复制密钥并将它添加到Gitee/Github的SSH配置中去。

四、验证连接

在本地终端执行

ssh -T git@gitee.com
